面向对象是一种编程思维，无关乎编程语言的具体实现，但有相关特性支持的语言更有利于将这种思维付诸于纸面。

## 对象和类

类（Class）是对象（Object）的抽象描述，对象是类的实例。

对象是一种实体，具有属性和行为。

类是一类对象的模板，描述一类对象的属性和行为。

例如以下是一个狗的类：

```text
狗：
    // 属性
    品种（variety）
    颜色（color）
    年龄（age）
    名字（name）
    // 行为
    进食（eat）
    活动（run）
    休息（sleep）
```

以下是这个类的几个对象

## 封装、继承与多态

### 封装

即隐藏对象的属性和具体实现，仅提供接口供外部访问和调用，编程语言对此的支持通常通过权限控制来实现。例如以下代码：

Java：
```java
public class Animal {
    private final String call;

    protected Animal(String call) {
        this.call = call;
    }

    public String getCall() {
        return this.call;
    }
}
```
以上`Java`代码中使用了三种权限控制符号，其中`public`代表对外公开，无论从何处都可访问，`private`代表私有，仅能在当前类中访问，`protected`代表保护，只能在当前类与其子类中访问。权限控制隐去了外部不需要关心或不想要外部访问的实现细节，降低了使用类时的复杂程度。

同样，在`C++`和`Python`中也存在类似的实现：

C++：
```cpp
class Animal {
private:
    std::string call;
protected:
    Animal(std::string call): call(std::move(call)) {}
public:
    std::string GetCall() {
        return this->call;
    }
};
```

Python：
```python
class Animal:
    __call: str

    def __init__(self, call: str):
        self.__call = call;

    def get_call(self):
        return self.__call;
```
特殊的，Python使用下划线`_`来进行访问权限标识，在字段和方法前添加双下划线`__`即标识该字段、方法为私有，单下划线`_`即标识该字段、方法为保护，无下划线即标识该字段、方法为公共。

### 继承

将多个类中相同的属性和方法再次提出，形成一个新的类，即父类，子类通过继承可以拥有父类的所有属性及方法。

如下代码所示：

Java：
```java
public class Animal {
    private final String call;

    protected Animal(String call) {
        this.call = call;
    }

    public String getCall() {
        return this.call;
    }
}
```
```java
public class Dog extends Animal {
    public Dog() {
        super("汪汪汪");
    }
}
```
```java
public class Cat extends Animal {
    public Cat() {
        super("喵喵喵");
    }
}
```
如上代码中的`Dog`和`Cat`类继承了`Animal`类，即`Dog`类与`Cat`类均为`Animal`的子类。在它们的构造函数中，使用了`super()`，该方法即为调用父类的构造函数并进行传参。由于父类中`getCall()`方法为`public`，所以可以由子类创建的对象调用到此方法。

C++：
```cpp
class Animal {
private:
    std::string call;
protected:
    Animal(std::string call): call(std::move(call)) {}
public:
    std::string GetCall() {
        return this->call;
    }
};
```
```cpp
class Dog: Animal {
public:
    Dog(): Animal("汪汪汪") {}
};
```
```cpp
class Cat: Animal {
public:
    Cat(): Animal("喵喵喵") {}
};
```

Python：
```python
class Animal:
    __call: str

    def __init__(self, call: str):
        self.__call = call;

    def get_call(self):
        return self.__call;
```
```python
class Dog(Animal):
    def __init__(self):
        super().__init__('汪汪汪')
```
```python
class Cat(Animal):
    def __init__(self):
        super().__init__('喵喵喵')
```
